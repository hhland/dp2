■前言

本压缩文件中保存的是《图解Java多线程设计模式》一书的示例程序。

■示例程序的目录结构

示例程序的目录结构如下所示：

    src
    +------ Future
    |       +------ Sample
    |       |       +       .
    |       |       |       .
    |       |       +------ Main.java
    |       |
    |       +------ jucSample
    |       |       +       .
    |       |       |       .
    |       |       +------ Main.java
    |       |
    |       +------ Q9-3
    |       |       +------ Main.java
    |       |
    |       +------ A9-3a
    |       |       +       .
    |       |       |       .
    |       |       +------ Main.java
    |       |
    |       +------ A9-3b
    |       |       +       .
    |       |       |       .
    |       |       +------ Main.java
    |
    +------ Balking
    |       +------ Sample
    |       |       .
    |       .       .
    .       .
    .

各个目录的意义如下。

    src/模式名/Sample      示例程序
    src/模式名/jucSample   示例程序(主要是使用了java.util.concurrent的示例程序)
    src/模式名/Q☆...      习题中的代码清单（☆表示习题编号）
    src/模式名/A☆...      习题解答中的代码清单（☆表示习题编号）
